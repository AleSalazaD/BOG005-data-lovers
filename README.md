### DATA LOVERS / DATA GHIBLI


### DEFINICION DE PRODUCTO
La página tiene como objetivo brindar información a los usuarios o a los visitantes sosbre las películas de Estudio Ghibli realizadas por

Para la realización de la página web 'Ghibli Lovers' realizamos un bosquejo inicial para entender cómo organizar y delimitar la información que proporcionaríamos, además de realizar decisiones de diseño en cuanto a espacios de caja, tipo de letra, tipos de íconos y paleta de colores.

Posteriormente utilizamos la tecnología de la plataforma Figma para diseñar un prototipo que se acercara más a la idea principal, donde además pudieramos escalar en diseño responsive, a partir de dos tamaños básicos (mobil y desktop).

Una vez realizados los prototipos procedimos a escribir el código base en HTML y CSS respondiendo a las ideas plasmadas en ellos.
